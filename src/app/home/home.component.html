The Sapphire Testnet

Ethereum Serenity Phase 0 Test Network

Run a Beacon Node Become a Validator

The Sapphire Testnet is a public network created by Prysmatic Labs implementing the Ethereum 2.0 Phase 0 protocol for a proof-of-stake blockchain enabling anyone holding test ETH to join and participate.

This is a highly experimental test network with no security guarantees. The network resets every 24 hours for improvements and maintenance as we help make eth2 a reality.

Frequently Asked Questions

What runs the network?

Built in Go and fully-open source, the network runs on an eth2 node implementation called Prysm. To participate in the network, anyone can send 3.2 ETH from the Goerli ETH1.0 testnet into a validator deposit contract, which will queue in the user as a validator in the system. Validators participate in proposing and voting on blocks in the protocol, and the network applies rewards/penalties based on their behavior.
